View on GitHub

Why Fork?

Forking lets you create your own copy of our project. Make changes, experiment, and submit improvements back to the community. It's open source collaboration at its best.

How to Contribute

1. Fork the repository

Click the Fork button on GitHub to create your own copy.

2. Make your changes

Clone locally and create a new branch for your feature.

git clone your-fork-url
git checkout -b feature-name

3. Submit a pull request

Push your changes and open a pull request for review.

git push origin feature-name
