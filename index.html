The First Blockchain Built on BNB

Our Innovation

We built the first blockchain on BNB by forking the official BNB GitHub repository. Chainze leverages the robust infrastructure and proven architecture of BNB Chain to create a new layer of blockchain innovation.

By utilizing BNB's open-source codebase, we've developed a custom blockchain implementation that maintains compatibility with the BNB ecosystem while introducing our own unique features and optimizations. This approach allows us to benefit from BNB's battle-tested consensus mechanism and security model.

Connect with MetaMask

Chainze is fully compatible with MetaMask, making it easy to import our blockchain and start transacting.

1. Open your MetaMask wallet
2. Click on the network dropdown at the top
3. Select "Add Network" and import using our RPC URL below
4. Save and start transacting on Chainze
